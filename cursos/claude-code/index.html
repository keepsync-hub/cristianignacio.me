Claude Code

Para personas que nunca, jamás, van a programar. Claude Code es un asistente que trabaja sobre archivos y tareas técnicas siguiendo tus instrucciones en lenguaje normal.

⏱️
Tiempo: 14 minutos · Nivel: experto No necesitas saber programar; sí tener ganas de probar algo nuevo.

🎯 Qué aprenderás

  • Qué es Claude Code y para qué sirve aunque no programes.
  • Qué tipo de tareas puede resolver por ti.
  • Cómo pedirle las cosas para que haga el trabajo técnico.

💡 La idea

Mucho "trabajo técnico" no es escribir código complejo: es ordenar archivos, automatizar una tarea repetitiva o limpiar datos. Claude Code hace eso siguiendo tus instrucciones, te muestra qué va a hacer y lo ejecuta con tu permiso.

🛠️ Paso a paso

  1. Explica el objetivo, no la técnica. "Renombra estas 200 fotos por fecha" en vez de pensar en comandos.
  2. Deja que proponga un plan. Te dirá qué hará antes de tocar nada.
  3. Aprueba paso a paso. Revisas y autorizas cada acción importante.
  4. Pide que te explique el resultado. Así entiendes qué pasó y puedes repetirlo.

📝 Cómo pedir una tarea

Objetivo: (qué quiero lograr, en palabras simples)
Sobre estos archivos: (cuáles y dónde están)
Antes de hacer cambios, explícame tu plan y pídeme
confirmación. No borres nada sin avisar.
Consejo: empieza con tareas reversibles y copias de tus archivos. Cuando le tomes confianza, sube la complejidad.

🚀 El resultado

Resuelves tareas que antes dependían de "alguien técnico": automatizas pasos repetitivos y ordenas tu trabajo digital, todo conversando en español.
