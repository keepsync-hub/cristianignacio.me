Claude Skills

Una Skill es un instructivo guardado que Claude activa solo cuando la tarea encaja. Lo creas una vez y lo llamas con un comando como /resumen o /correo, sin repetir nunca el contexto.

⏱️
Tiempo: 6 minutos · Nivel: intermedio Perfecto para tareas que repites con el mismo formato.

🎯 Qué aprenderás

  • Qué es una Skill y en qué se diferencia de un buen prompt.
  • Cómo crear la primera, paso a paso.
  • Trucos para que se active justo cuando debe.

💡 La idea

Hay tres formas de darle contexto a Claude: un mensaje muy largo, un archivo que subes, o un Proyecto que abres. Las Skills van un paso más: se disparan solas. Claude reconoce la tarea y activa la Skill correcta sin que tengas que pedírselo.

🛠️ Paso a paso

  1. Elige una tarea repetitiva con un resultado siempre parecido (un resumen, un correo, un reporte).
  2. Usa el creador de Skills. En Cowork, pídele que te ayude a crear una Skill para esa tarea y agrega "pregúntame primero".
  3. Responde la entrevista con detalle. "Escribo reportes" no sirve. "Escribo reportes semanales que empiezan con la cifra clave, usan tres secciones y cierran con próximos pasos" sí funciona.
  4. Guárdala y pruébala. Instálala y verifica que se active con distintas formas de pedir lo mismo, y que no se active cuando no toca.

📝 Estructura de una Skill

Nombre: (cómo la vas a llamar)
Cuándo usarla: (en qué situación)
Cuándo NO usarla: (para que no se active de más)
Objetivo: (qué debe lograr)
Pasos: 1) ... 2) ... 3) ...
Formato de salida: (cómo debe verse el resultado)
Ejemplo: (un caso bien resuelto)

✨ Trucos que ayudan

  • La línea de "cuándo NO usarla" importa más que la de "cuándo sí": evita que se active donde no corresponde.
  • Si no se dispara, pregúntale a Claude cuándo usaría esa Skill; verás si la descripción está vaga.
  • Las Skills se combinan con tu archivo de estilo: una pone el tono, la otra el proceso.
  • Producen un buen punto de partida, no la perfección. Casi siempre conviene revisar.
Una buena Skill incluye un ejemplo de resultado correcto. Es lo que más sube la calidad y la consistencia.

🚀 El resultado

Tienes tu primera Skill: un instructivo reutilizable que convierte una tarea repetitiva en algo de un comando, siempre con tu mismo estándar.
