Claude Skills

Una Skill es un instructivo reutilizable que le enseña a Claude a hacer una tarea exactamente como tú la haces. La creas una vez y la invocas siempre que la necesites.

⏱️
Tiempo: 6 minutos · Nivel: intermedio Perfecto para tareas que repites con el mismo formato.

🎯 Qué aprenderás

  • Qué es una Skill y para qué tareas vale la pena.
  • Cómo escribir tu primera Skill, paso a paso.
  • Cómo probarla y mejorarla con el uso.

💡 La idea

Si cada semana le explicas a Claude lo mismo ("redacta el resumen así, con este formato"), eso es candidato a Skill. En vez de repetir el instructivo, lo guardas una vez y Claude lo sigue cada vez que la activas.

🛠️ Paso a paso

  1. Elige una tarea repetitiva y con un resultado siempre parecido (un resumen, un correo, un reporte).
  2. Escribe el instructivo. Explica el objetivo, los pasos y el formato exacto de salida.
  3. Guárdala como Skill con un nombre claro que recuerdes.
  4. Pruébala y ajústala. Úsala con un caso real y corrige el instructivo donde falle.

📝 Estructura de una Skill

Nombre: (cómo la vas a llamar)
Cuándo usarla: (en qué situación)
Objetivo: (qué debe lograr)
Pasos: 1) ... 2) ... 3) ...
Formato de salida: (cómo debe verse el resultado)
Ejemplo: (un caso bien resuelto)
Consejo: una buena Skill incluye un ejemplo de resultado correcto. Es lo que más sube la calidad y la consistencia.

🚀 El resultado

Tienes tu primera Skill: un instructivo reutilizable que convierte una tarea repetitiva en algo de un clic, siempre con tu mismo estándar.
