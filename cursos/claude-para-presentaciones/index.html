Claude para presentaciones

Crear una presentación desde cero da pereza. Con Claude pasas de un tema suelto a una estructura sólida y unas diapositivas claras, sin pelearte con el diseño.

⏱️
Tiempo: 7 minutos · Nivel: intermedio Sirve para reuniones, clases, pitch de ventas o informes.

🎯 Qué aprenderás

  • Cómo planear la presentación antes de tocar las diapositivas.
  • Cómo pedir el contenido diapositiva por diapositiva.
  • Cómo dejarla lista para tu herramienta de slides.

💡 La idea

Una buena presentación es primero una buena historia. En vez de empezar eligiendo plantillas, deja que Claude te ayude a ordenar el mensaje y luego a redactar cada lámina. El diseño es lo último.

🛠️ Paso a paso

  1. Define objetivo y público. Qué quieres que pase al terminar y a quién le hablas.
  2. Pide el guion. Una estructura de 6–10 diapositivas con el título y la idea principal de cada una.
  3. Desarrolla lámina por lámina. Pídele el texto breve y las notas del orador de cada diapositiva.
  4. Llévalo a tus slides. Pega el contenido en tu herramienta y aplica tu plantilla.

📝 Prompt para el guion

Ayúdame a preparar una presentación.
Tema: (de qué trata)
Objetivo: (qué quiero lograr con el público)
Público: (quiénes son)
Duración: (cuántos minutos)

Dame primero la estructura: 8 diapositivas con título
e idea principal de cada una. No desarrolles aún.
Consejo: una idea por diapositiva. Si una lámina necesita tres viñetas densas, probablemente son tres diapositivas.

🚀 El resultado

Tienes una presentación con una historia clara, textos breves y notas para hablar, lista para vestir con tu plantilla y presentar con seguridad.
