📈 Reporte automático de precios para carga en CRM

Genera un Excel limpio con las actualizaciones de precios de tu proveedor mediante una rutina de Claude Code y automatiza su carga en la interfaz de tu CRM usando la extensión Claude CoWork.

⏱️
Tiempo: 45 minutos · Dificultad: intermedio Necesitas Claude Code y la extensión de Chrome Claude CoWork.

🧩 El problema

Tu proveedor actualiza sus precios constantemente y necesitas reflejar esos cambios en tu CRM para cotizar correctamente. El problema es que tu CRM es cerrado: no permite conectar herramientas externas ni Inteligencia Artificial para que se actualice solo. Hacer el seguimiento, buscar los cambios y armar el archivo a mano todas las semanas te quita tiempo valioso y abre la puerta a errores de digitación.

💡 La idea

Crear un flujo automatizado con una rutina de Claude Code que se ejecute en segundo plano de forma programada para extraer los precios actualizados a un Excel limpio. Después, usar Claude CoWork desde su extensión de Google Chrome para automatizar la carga de esos datos directamente en la interfaz del CRM, superando la limitación de un sistema cerrado.

🛠️ La construcción, paso a paso

  1. Configura la rutina de Claude Code en segundo plano para que revise las actualizaciones del proveedor de forma programada.
  2. Programa la extracción. Que genere un archivo de Excel limpio y ordenado todos los lunes exactamente a las 9:00 AM.
  3. Instala Claude CoWork (extensión de Google Chrome) en tu navegador.
  4. Configura la carga. Que Claude CoWork tome los datos del Excel generado y rellene de forma secuencial los campos en la interfaz de usuario del CRM.

📝 El prompt / los archivos que usamos

Todo el material está en la carpeta archivos/:

  • Rutina de extracción para Claude Code: archivos/rutina_claude_code.js
  • Prompts listos para copiar: archivos/prompt-rutina-claude-code.md
  • Ejemplo de Excel limpio para la extensión: archivos/plantilla_precios.csv
📂
Material de esta solución El Excel final (plantilla_precios.xlsx) lo genera la propia rutina; aquí dejamos un ejemplo en CSV con las mismas columnas para que veas cómo queda.

🚀 El resultado

Un reporte de precios exacto y puntual listo cada lunes, junto con una carga de datos al CRM completamente manos libres que elimina los errores humanos de digitación y evita el trabajo manual aburrido.

🔁 Hazlo tuyo

  • Adapta la rutina para que extraiga datos de varios proveedores a la vez.
  • Modifica la frecuencia (por ejemplo, diaria en lugar de semanal) si tu mercado requiere actualizaciones en tiempo real.
