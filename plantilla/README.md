# 🧱 Plantilla de solución — Claude desde Cero

Copia esta carpeta para crear una solución nueva. El objetivo es contar **cómo
se construyó**, en lenguaje claro y sin tecnicismos.

## Cómo usar la plantilla

1. Duplica la carpeta `plantilla/` dentro de `soluciones/`.
2. Renómbrala con un nombre corto en minúsculas y con guiones.
   Ejemplo: `soluciones/resumir-un-pdf/`.
3. Rellena el `README.md` (para GitHub) y el `index.html` (para la web).
4. Pon cualquier material de apoyo dentro de `archivos/`.
5. Añade una tarjeta enlazando tu solución en `soluciones/index.html` y, si
   quieres, en la portada `index.html`.

---

## 📄 Plantilla de contenido (copia desde aquí)

```markdown
# [emoji] Título corto y claro

> **Categoría:** ___ · **Dificultad:** Principiante/Intermedio · **Tiempo:** __ min

Una frase que explique qué consigue esta solución.

## 🧩 El problema
¿Qué necesidad real resuelve? Cuéntalo como a un amigo.

## 💡 La idea
La estrategia en una o dos frases.

## 🛠️ La construcción, paso a paso
1. Paso uno.
2. Paso dos.
3. Paso tres.

## 📝 El prompt / los archivos que usamos
Pega aquí el prompt o enlaza los archivos de la carpeta `archivos/`.

## 🚀 El resultado
Qué obtienes y cómo se ve.

## 🔁 Hazlo tuyo
Ideas para adaptarlo.

## 🎓 Cursos recomendados
Los 2-3 cursos esenciales para implementar esta solución, cada uno con una
frase de por qué se necesita. Enlázalos a https://cristianignacio.me/cursos/<NN-slug>/.
```

## ✅ Checklist antes de publicar

- [ ] Incluye el problema, la construcción y el resultado.
- [ ] El material (prompts, archivos) está en `archivos/`.
